# 🔐 Random Password Generator in Python

This is a simple Python project that generates strong and secure random passwords.

## 🚀 Features
- Generates random passwords
- Includes:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (!@#$...)
- User can choose password length
- Ensures strong password security

## 🛠️ Technologies Used
- Python
- Built-in modules: random, string

## ▶️ How to Run

1. Make sure Python is installed
2. Run the following command:

```bash
python password.py
